# 法式奶油风（French Cream）室内设计规范

## 核心理念
法式奶油风源自巴黎传统公寓美学，融合现代简约与古典优雅。核心要素：柔和曲线、温暖中性色调、精致而不刻意的装饰感。

## 色彩体系
- 主色调：奶油白、象牙色、米色、浅杏色
- 辅助色：拿铁色、奶茶色、淡灰粉
- 点缀色：金色（灯饰/五金）、深绿（植物）、黑色（线条勾勒）
- 色彩过渡：避免强烈对比，所有色彩之间过渡要柔和

## 材质选择
- 墙面：乳胶漆（哑光）、石膏线装饰、护墙板（法式线条）
- 地面：鱼骨拼木地板、人字拼木地板（浅橡木色）
- 天花板：石膏灯盘、简约吊顶线条
- 家具：绒面沙发、大理石台面茶几、黄铜五金件
- 织物：天鹅绒、羊绒、亚麻混纺

## 造型特征
- 拱门元素：门洞、壁龛、镜子采用弧形拱门造型
- 线条装饰：墙面石膏线（宽度 8-12cm）、踢脚线加高至 15-20cm
- 家具曲线：沙发扶手、椅背、床头板带柔和弧线
- 壁炉造型：即使无真壁炉，也保留法式壁炉架作为视觉焦点

## 灯光设计
- 色温：3000K-3500K 暖白光
- 主灯：水晶吊灯或黄铜吊灯（直径与房间对角线比例 1:10）
- 辅助光源：黄铜壁灯、陶瓷台灯
- 氛围灯：隐藏式灯带（天花凹槽、电视背景墙后）
- 避免冷白光和工业风裸露灯泡

## 软装元素
- 欧式石膏雕像、复古相框
- 鲜花：绣球、玫瑰、郁金香（避免假花）
- 地毯：素色或低饱和花纹的羊毛地毯
- 窗帘：奶油色丝绒或棉麻，落地且带自然褶皱
- 镜子：金属边框圆镜或拱形落地镜

## 常见空间应用
- 客厅：弧形沙发 + 大理石茶几 + 石膏线电视墙 + 黄铜吊灯
- 卧室：绒面床头板 + 鱼骨拼地板 + 奶油色床品 + 不对称床头灯
- 餐厅：大理石餐桌 + 藤编餐椅 + 黄铜吊灯 + 拱形壁龛
