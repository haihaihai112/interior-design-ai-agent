# 室内灯光色温与照明设计标准

## 色温标准（Kelvin / K）
- 2700K：超暖黄光 —— 卧室、休息区、侘寂风、中式风格
- 3000K：暖黄光 —— 客厅、餐厅、法式奶油风、日式风格
- 3500K：暖白光 —— 厨房、卫生间、书房
- 4000K：自然白光 —— 办公室、工作室、极简现代风
- 5000K+：冷白光 —— 不推荐用于家装，仅适用于医院/实验室

## 照度标准（Lux / 勒克斯）
- 客厅整体：100-200 Lux
- 阅读区：300-500 Lux
- 厨房操作台：500-700 Lux
- 卧室整体：50-100 Lux
- 卫生间化妆区：400-600 Lux
- 走廊/过道：50-100 Lux

## 灯光层次设计（三层法则）
1. 环境光（Ambient）：提供整体均匀照明，占总光量 20-30%
   - 天花灯带、间接照明、筒灯（均匀分布）
2. 重点光（Task/Accent）：照亮特定区域或物体，占总光量 40-50%
   - 射灯照画、吊灯照餐桌、阅读落地灯
3. 装饰光（Decorative）：视觉焦点和氛围营造，占总光量 20-30%
   - 壁灯、台灯、灯带、纸灯

## 灯具选型参数
- 光束角（Beam Angle）：
  - 15°-25°：窄光束，用于重点照明（照画/雕塑）
  - 30°-45°：中光束，用于局部照明（餐桌/岛台）
  - 60°+：宽光束，用于基础照明（走廊/整体）
- 显色指数（CRI / Ra）：家居 ≥ 90，博物馆级 ≥ 95
- 防眩光：UGR < 19（统一眩光值）

## 无主灯设计要点
- 不安装中央吸顶灯/吊灯
- 替代方案：嵌入式筒灯 + 磁吸轨道灯 + 线性灯带
- 筒灯间距：天花板高度的 1-1.5 倍
- 筒灯距墙：30-50cm（洗墙效果最佳）
- 特别注意：无主灯设计需要在天花板阶段预埋线路

## 各空间灯光方案速查
- 20平米客厅：2 条磁吸轨道灯 + 3 盏明装筒灯 + 1 盏落地灯 + 电视背景灯带
- 15平米卧室：床头壁灯 ×2 + 天花灯带（间接照明）+ 衣柜内感应灯
- 10平米厨房：操作台上方灯带 + 3 盏嵌入式筒灯 + 水槽上方射灯
- 8平米卫生间：镜前灯（左右两侧各一）+ 淋浴区防水筒灯 + 天花灯带
