# 海外设计趋势与内容发布参考

## 趋势观察渠道
- Pinterest：适合观察空间风格、色彩组合、软装搭配和关键词热度。
- Instagram：适合观察设计工作室、家居品牌、生活方式账号的视觉表达。
- Dezeen / ArchDaily：适合观察建筑与室内设计行业案例。
- Behance：适合观察视觉包装、作品集呈现方式和图文排版。
- TikTok / YouTube Shorts：适合观察短视频叙事、前后对比和工具教程类内容。

## 近期常见方向
- Japandi / Wabi-sabi：自然材质、低饱和色、留白、手工痕迹。
- Quiet luxury：低调高级、天然石材、皮革、暖灰和柔和金属。
- French cream：奶油色、弧线家具、石膏线、黄铜灯具、温柔复古感。
- Warm minimalism：极简但不冷，强调木材、织物、灯光层次。
- Biophilic design：室内绿植、自然光、木石材、室内外连接。

## 内容发布结构
### 小红书 / Instagram 图文
1. 标题：风格 + 空间 + 记忆点。
2. 首图：最完整、最有氛围的一张效果图。
3. 正文：设计亮点、材质清单、灯光说明、适合人群。
4. 标签：风格标签、空间标签、工具标签、趋势标签。

### 短视频脚本
1. 0-2 秒：展示最终效果图或前后对比。
2. 3-6 秒：说明用户需求或设计痛点。
3. 7-15 秒：展示材质、灯光、家具选择过程。
4. 16-25 秒：展示多角度效果图。
5. 结尾：总结风格关键词和可复用搭配公式。
